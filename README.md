# 🎰 PYC888 Slot Machine

A retro-style slot machine game built with Python and Tkinter.

This project features animated spinning reels, a betting system, balance tracking, and randomized slot machine outcomes inspired by classic casino games.

---

## Features
- GUI built with Tkinter
- Randomized slot machine reels
- Betting system
- Balance tracking
- Jackpot rewards
- Animated spinning reels
- Retro casino theme
- Color-changing title animation

---

## Technologies Used
- Python
- Tkinter
- Random
- Time
